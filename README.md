# SubjectiveModeling

Мини-библиотека на Haskell: категорный матаппарат теории субъективного моделирования
Ю.П. Пытьева и смежных теорий неопределённости (возможность, дискретная вероятность,
нечёткая логика) на общем ядре — кванталях, расширениях Кана и монаде распределений
над полукольцом.

> Источник теории: Ю.П. Пытьев, «Математические методы субъективного моделирования
> в научных исследованиях», Вестник МГУ, Сер. Физика. Астрономия, 2018, №1–2.

## Идея

Одна и та же математика порождает разные теории неопределённости заменой алгебры значений:

| Полукольцо / квантале | Теория |
|---|---|
| `([0,1], max, min)` | возможность / правдоподобия Пытьева (Pl) |
| `([0,1], +, ×)` | дискретная вероятность |
| `(Bool, ∨, ∧)` | классическая логика и достижимость |

- `bindD` монады распределений над `(max, min)` — это **pl-интеграл** Пытьева
  (Теорема 1.1), над `(+, ×)` — формула полной вероятности, над `(∨, ∧)` — достижимость.
- Мера правдоподобия — **левое расширение Кана** вдоль профунктора принадлежности:
  `Pl = Lan_J τ`; мера доверия — **правое** вдоль профунктора дополнения: `Bel = Ran_{θJ} τ̄`.
- Условное распределение — **residuation** квантали: `τ(z₁|z₂) = τ(z₂) ⊸ τ(z₁,z₂)`
  (правый сопряжённый к `min(−, a)`), нормировка без субъективных шкал.
- Над `Bool` всё вырождается в логику: `Pl = ∃`, `Bel = ∀`.

## Модули

| Модуль | Содержимое |
|--------|-----------|
| `src/Quantale.hs` | Классы `ApproxEq` / `Lattice` / `Quantale`; инстансы `UnitInterval` и `Bool`; инволюция `theta`; группа автоморфизмов Γ; проверки законов (residuation-сопряжение, дистрибутивность фрейма) |
| `src/KanExtension.hs` | `lanAlong` / `ranAlong` вдоль профункторов; `plMeasure` / `belMeasure`; обогащённые категории (метрики Ловера), Йонеда-пополнение `yonedaHat`; двойственность Исбелла `isbellO ⊣ isbellSpec` |
| `src/Bitopos.hs` | Топологии Скотта на `[0,1]` и индуцированная битопология на X; интервальный билатис `[Bel, Pl]` с порядками истинности и информации (Белнап FOUR на континууме) |
| `src/Distribution.hs` | `class Semiring`; монада распределений `Dist r a`; `Poss` (возможность), `ProbW` (вероятность), `Bool` (достижимость); ядра Клейсли, законы монады |
| `src/SubjectiveModel.hs` | Слой Пытьева: `SubjModel`, меры и интегралы, независимость, residuation-кондиционирование, энтропии, идентификация состояний, комбинирование экспертных и эмпирических данных |

Зависимости — только сверху вниз; ядро (`Quantale`, `KanExtension`, `Distribution`)
полиморфно, прикладной слой (`Bitopos`, `SubjectiveModel`) конкретен на `[0,1]`,
где инволюция θ и энтропии содержательны. В каждом модуле — блок `-- ИДЕИ РАСШИРЕНИЯ`
(Лукасевич, тропические квантали, полукольцо Витерби, Isbell envelope и пр.).

## Сборка, тесты, примеры

Полноценный cabal-пакет (зависимости: только `base` и `containers`):

```bash
cabal build
cabal test                                  # 33 численные проверки законов
cabal run possibility-example               # теория возможностей
cabal run subjective-modeling-example       # субъективное моделирование Пытьева
```

Без cabal (достаточно ghc):

```bash
ghc -isrc test/Spec.hs -o spec && ./spec
ghc -isrc examples/PossibilityExample.hs -o ex1 && ./ex1
ghc -isrc examples/SubjectiveModelingExample.hs -o ex2 && ./ex2
```

Тесты проверяют: residuation-сопряжение и дистрибутивность фрейма (для `[0,1]` и `Bool`),
дуальность θ, Pl/Bel против прямых формул на всех подмножествах, дуальную согласованность,
Йонеда-пополнение (наименьший пресноп, идемпотентность), единицу и треугольник Исбелла,
законы монады над тремя полукольцами, сохранение нормировок, уравнение кондиционирования
Пытьева (решение + максимальность + нормировка), энтропии, правило идентификации,
комбинирование экспертов и законы интервального билатиса с interlacing.

### Примеры

- [`examples/PossibilityExample.hs`](examples/PossibilityExample.hs) — **теория
  возможностей**: одна цепь переходов над тремя полукольцами (возможность /
  вероятность / достижимость), стабилизация sup-min степеней, нечёткая логика
  как квантале (t-норма = qTensor, импликация Гёделя = residuation).
- [`examples/SubjectiveModelingExample.hs`](examples/SubjectiveModelingExample.hs) —
  **субъективное моделирование**: диагностика двигателя по Пытьеву — экспертные
  правдоподобия, меры Pl/Bel, кондиционирование по наблюдению через residuation
  (с автоматической нормировкой), комбинирование эксперта с эмпирикой,
  энтропии, оптимальное правило идентификации, неразличимость состояний как
  обогащённый hom и tight span Исбелла.

## Использование

GHCi:

```
ghci src/Quantale.hs src/KanExtension.hs src/Bitopos.hs src/Distribution.hs src/SubjectiveModel.hs
```

IHaskell (Jupyter) — одной командой в setup-ячейке (`:load` сбрасывает импорты,
поэтому все `import` после него):

```
:load src/Quantale.hs src/KanExtension.hs src/Bitopos.hs src/Distribution.hs src/SubjectiveModel.hs
import Quantale
import KanExtension
import Distribution
import SubjectiveModel
```

Пример:

```haskell
-- субъективная модель с дуально согласованными мерами
let sm = dualConsistent "abc" (\c -> case c of { 'a' -> 1.0; 'b' -> 0.6; _ -> 0.2 })
smPl sm "ab"          -- 1.0  (Pl = Lan вдоль принадлежности)
smBel sm "ab"         -- 0.8  (Bel = Ran вдоль дополнения)

-- кондиционирование = residuation (максимальное решение уравнения Пытьева)
condTau dom tauJoint z2 z1

-- одна монада, три теории
nStepsD 3 stepPoss Sun   -- sup-min цепь (возможность)
nStepsD 3 stepProb Sun   -- марковская цепь (вероятность)
nStepsD 3 stepBool Sun   -- достижимость

-- классическая логика как вырождение
plMeasure [1,2,3] (> 1) [2,3]   -- True  (∃)
belMeasure [1,2,3] (> 1) [2,3]  -- False (∀ вне E)
```

Зависимости: только `base` и `containers`. Без расширений GHC сверх стандартных.

Библиотека извлечена из учебных IHaskell-ноутбуков курса «Haskell + теория категорий»,
где все законы (residuation, монада над тремя полукольцами, единица и треугольник
Исбелла, interlacing билатиса) проверяются численно при каждом прогоне.
